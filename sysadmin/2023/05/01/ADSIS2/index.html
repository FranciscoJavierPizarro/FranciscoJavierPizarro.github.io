TFG

Este TFG fue mi proyecto final para graduarme como ingeniero informático, este trata sobre el diseño, implementación y explotación de un sistema SOC basado en software OpenSource. Tuvo una primera etapa de administrador de sistemas en la que se puso en marcha la distinta infraestructura(MVs, SIEM, SOAR, EDR, Firewall...). Posteriormente tuvo una segunda fase en la que desde la perspectiva de un Blue Team se implantarón y probaron los distintos CDUs.

  • Client: NOLOGIN
  • Date: Agosto 2023
  • Service: Ciberseguridad

ADSIS2

En la primera parte se realizo el despliegue de una red con todo lo necesario para una empresa hoy en día desde lo más básico partiendo de la red, VLANs y firewalls hasta los elementos más avanzados como el directorio activo y el sistema de ficheros en red. En la segunda parte se trabajo todos los aspectos de HA de K8s, montando al final sobre este un sistema de almacenamiento distribuido Ceph, que empleaban una página web y un registro de contenedores, aquí se trabajo también la IaC con Terraform y el provisionamiento a alto nivel con Puppet y Ansible.

  • Client: Unizar
  • Date: Mayo 2023
  • Service: SysAdmin

REIGN CHESS

En este proyecto llevado a cabo por un grupo de 7 personas, se realizo un desarrollo completo de software como si de una empresa real se tratase, teniendo un cliente al que satisfacet. El proyecto cuenta con un Backend común desarrollado en ExpressJS, un frontend web desarrollado en ReactJS y una aplicación móvil desarrollada en Flutter. Este es un juego de ajedrez con distintas mecánicas tales como jugar contra una IA, jugar torneos, jugar partidas privadas, jugar competitivo, poder pausar y reanudar partidas. Fue una gran experiencia de aprendizaje.

  • Client: REIGN
  • Date: Abril 2023
  • Service: Programador
